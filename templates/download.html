Installation Instructions

  1. Install Python 3.7+
    Download from python.org
  2. Download and extract the client package
    Use the download button above
  3. Install dependencies
    pip install -r client_requirements.txt
  4. Run the client
    python client_app.py
  5. Connect to server
    Enter your server URL (e.g., http://localhost:5000) and click Connect
  6. Select audio device
    Choose your system audio output or WASAPI loopback device
  7. Start recording
    Click "Start Recording" to begin system audio transcription

Troubleshooting

  • No audio devices shown: Try running as administrator or check audio drivers
  • Can't capture system audio on macOS: Install audio routing software like Loopback
  • Connection failed: Ensure the web app server is running and accessible
  • Poor audio quality: Check audio device settings and sample rate
